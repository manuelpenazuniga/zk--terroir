Pricing

Premium payout

Ed25519 public key that receives the premium.

Must be exactly 64 hexadecimal characters (a 32-byte public key).

Advanced — lot identity (lot_id, season_id, lot_secret)

Changing lot_secret or season_id moves this lot outside the accredited set seeded on-chain (r_cert = 975646…302554). The proof will still be generated, but the contract will reject it.

Private by design: the witness and the proof are computed locally with snarkjs.groth16.fullProve — your inputs never leave this device. The contract only ever sees the proof.

Proving pipeline

  1. 1Poseidon init
  2. 2Witness
  3. 3Prove (Groth16)
  4. 4Verify (VK)
  5. 5Encode (Soroban)
